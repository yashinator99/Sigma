# Welcome to Project 0

## How to run it:

- Open git bash
- Go to the folder that has app.py
- call python app.py
- Go to this this link http://127.0.0.1:5000
- Login using login info or sign up
- in login you can request, view request, or delete your account

## Technologies:

- Python Flask
- Postman
- DBeaver
- AWS RDS
